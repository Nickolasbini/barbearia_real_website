# 🚀 Site de Alta Conversão para Negócio Local: Barbearia Real

Este é o projeto de uma **Landing Page de alta conversão** voltada para negócios locais (utilizando uma barbearia vintage como exemplo). O grande diferencial desta estrutura é a **fricção zero**: o site é 100% estático, leve, responsivo e utiliza a API nativa do WhatsApp com mensagens pré-preenchidas e codificadas (`URL Encoding`) para direcionar o cliente direto para o agendamento de serviços específicos, eliminando a necessidade de infraestruturas complexas de backend.

O projeto foi desenvolvido para o canal **CódigoPrático** e serve como um modelo perfeito para quem deseja monetizar prestando serviços de desenvolvimento para empresas da sua região.

## 📱 Funcionalidades

* ⚡ **Performance Extrema:** Construído puramente com HTML5 e Tailwind CSS via CDN (carregamento instantâneo).
* 🎯 **Foco em Conversão:** CTAs estratégicos espalhados pelas seções principais (Hero, Serviços, Rodapé e Botão Flutuante).
* 📱 **Menu Mobile Nativo:** Menu responsivo controlado com JavaScript puro (`Vanilla JS`), sem dependências externas.
* 🔗 **Links Dinâmicos (WhatsApp API):** Cada botão de serviço já abre a conversa no WhatsApp do estabelecimento com o texto do serviço escolhido pré-configurado.
* 🎨 **Design Premium:** Identidade visual customizada (Charcoal, Cream e Gold) usando fontes do Google Fonts (*Oswald* e *Inter*).
* 📈 **SEO & Open Graph:** Configuração básica de meta tags para indexação no Google e visualização profissional com card de imagem ao compartilhar o link no WhatsApp/Instagram.
* 💓 **Efeito de Pulso CSS:** Botão flutuante do WhatsApp com animação de pulso nativa para atrair o clique do usuário.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** — Estruturação semântica.
* **Tailwind CSS (via CDN)** — Estilização rápida, utilitária e responsiva.
* **JavaScript (Vanilla)** — Manipulação do DOM para o menu mobile.
* **Google Fonts** — Fontes *Oswald* (títulos) e *Inter* (corpo do texto).
